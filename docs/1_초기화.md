#### `glutInit(&argc, argv)`

##### OpenGL을 사용하는 프로그램에서 GLUT 기반 윈도우를 만들기 전에 반드시 호출해야 하는 필수 초기화 함수
* GLUT 라이브러리 초기화 및 윈도우 시스템 연결
  * GLUT 라이브러리 초기화: GLUT 라이브러리를 시작하고 
  * 운영체제의 윈도우 시스템과 (컨텍스트?)연결을 설정합니다.

* 프로그램의 커맨드라인 인자(argc, argv)에서 GLUT 관련 옵션을 찾아서 제거하고 처리합니다.
* 포인터 형식 &argc로 전달되므로, 함수 내에서 GLUT 옵션을 제거 후 argc 값이 갱신됨
* 오류 처리: 초기화 실패 시(윈도우 시스템 연결 불가, OpenGL 미지원 등) 프로그램이 에러 메시지와 함께 종료됨


#### `glutInitWindowPosition & glutInitWindowSize`

##### glutCreateWindow()로 윈도우를 생성하기 전에 호출되어 초기 윈도우 설정을 요청

|함수|역할|기본값|
|---|---|---|
|glutInitWindowPosition(x, y)|화면의 좌상단으로부터 윈도우 위치 지정|(-1, -1)|
|glutInitWindowSize(w, h)|윈도우 가로×세로 크기 지정|300×300|

#### `glutInitDisplayMode()`

* glutCreateWindow() 이전에 호출
* 색상 모드, 버퍼링 방식, 추가 버퍼 등을 비트마스크로 조합
* 생성될 모든 윈도우에 적용됨

##### 주요 Display Mode

1. 색상 모드 (중 1개 선택)
    |플래그|설명|사용 사례|
    |---|---|---|
    |GLUT_RGBA|R, G, B, A 각각 독립 (기본값)|현대 그래픽 (권장)|
    |GLUT_INDEX|-|-|
    |GLUT_LUMINANCE|-|-|

2. 버퍼링 방식 (중 1개 선택)
    |플래그|설명|장점/단점|
    |---|---|
    |GLUT_SINGLE|단일 버퍼 (기본값)|메모리 적음, 깜빡임 발생 ❌|
    |GLUT_DOUBLE|더블 버퍼|부드러운 애니메이션 ✅|
    * 더블 버퍼링
    ```
    0. Front Buffer에 그려진 상태 교체 직전
    1. Back buffer에 그리기
    2. glutSwapBuffers() 호출
    3. Front/Back 교환
    4. 다음 프레임 반복
    ```

3. 추가 버퍼 (선택, 비트 OR로 조합)
    |플래그|크기|용도|
    |---|---|---|
    |GLUT_ALPHA|8비트/픽셀|투명도 채널 필요 시|
    |GLUT_DEPTH|24-32비트|3D 렌더링 (Z-buffer)|
    |GLUT_STENCIL|8비트|스텐실 마스킹 (고급 효과)|
    |GLUT_ACCUM|16-32비트/채널|모션 블러, 앤티에일리싱|
    |GLUT_MULTISAMPLE|가변|부드러운 경계선 (MSAA)|
    |GLUT_STEREO|가변|3D 안경 렌더링|
    ```cpp
    // GLUT_DOUBLE : 더블 버퍼링 사용.
	// 	화면에 보이는 front buffer와 그리는 back buffer 두 개를 두고,
	// 	그리기 완료 후 glutSwapBuffers()로 교체. 깜빡임(flickering) 방지
	// GLUT_RGBA : RGBA 색상 모드 사용 (반투명 오브젝트 렌더링 가능)
	// GLUT_DEPTH : GLUT_DEPTH 추가 = Z-buffer 활성화 3D 물체의 깊이 정보 저장
	glutInitDisplayMode(GLUT_DOUBLE | GLUT_RGBA | GLUT_DEPTH);
    ```

#### `glutCreateWindow()` 

##### OpenGL 렌더링을 위한 윈도우를 실제로 생성하는 함수

* 호출 시점에 다음 작업이 순차적으로 수행됨:
    1. 운영체제에 윈도우 요청 (크기, 위치는 이전 설정 참조)
    2. 각 윈도우마다 독립적인 OpenGL Context 생성
    3. 새로운 윈도우가 현재 윈도우(Current Window) 로 설정
    4. 고유한 정수 식별자 반환

##### 리턴값의 의미 (핵심!)
* 윈도우 고유 식별자 (Window ID)
    ```
    int win1 = glutCreateWindow("Window 1");
    glClearColor(1.0, 0.0, 0.0, 1.0);  // 윈도우1: 빨강

    int win2 = glutCreateWindow("Window 2");
    glClearColor(0.0, 0.0, 1.0, 1.0);  // 윈도우2: 파랑

    // 각 윈도우는 독립적인 OpenGL 상태 유지
    ```
